# minesweeper-cpp
Simple minesweeper game written in C++ with NCurses!
# Requirements
## For using
You need just ncurses.
```bash
$ sudo pacman -S ncurses
```
## For build
You need clang, make and ncurses
```bash
$ sudo pacman -S clang make ncurses
```

# How to build
Just type
```bash
$ make
```
# How to launch
Run Minesweeper file
```bash
$ ./Minesweeper
```
